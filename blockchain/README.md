## üöÄ Quick Start

### Prerequisites
- Node.js 18+ with npm/yarn
- GCP credentials configured for us-west1 region
- SallyPort authentication token
- Blockchain RPC endpoints configured

### Basic Setup
```bash
# Install dependencies
npm install

# Configure environment
cp .env.sample .env
# Edit .env with your configuration

# Launch BACA Coin (testnet)
./launch-baca-coin.sh --network testnet --region us-west1

# Deploy Smart Contracts
./deploy-smart-contracts.sh --env development --verify

# Mint Achievement NFT
./mint-nft.sh --type pilot-badge --agent RIX-001 --achievement mastery-33
```
